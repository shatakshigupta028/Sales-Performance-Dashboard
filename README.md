# Pizza Sales Performance Dashboard (SQL + Power BI)

## 📌 Overview

This project analyzes a 2015 pizza sales dataset using SQL and Power BI. It covers the full process from data preparation to creating dashboards and insights.

The goal is to understand sales performance, customer behavior, and product trends.

---

## 🛠️ Tools & Technologies

* **SQL (MS SQL Server)** – Data extraction, transformation, and KPI calculations
* **Power BI** – Data visualization and dashboard creation
* **Power Query** – Data cleaning and transformation
* **DAX** – Measures and calculated insights

---

## 📂 Project Workflow

### 1. Data Import & Preparation

* Imported raw CSV dataset into **MS SQL Server**
* Structured and cleaned data for analysis

---

### 2. SQL Data Analysis

#### 🔹 KPI Calculations

* Total Revenue
* Average Order Value
* Total Pizzas Sold
* Total Orders
* Average Pizzas per Order

#### 🔹 Trend Analysis

* Daily sales trends
* Monthly performance trends

#### 🔹 Sales Breakdown

* Percentage of sales by:

  * Pizza Category
  * Pizza Size

#### 🔹 Performance Analysis

* Top 5 best-selling pizzas
* Bottom 5 worst-selling pizzas

---

### 3. Power BI Dashboard

* Connected Power BI to SQL Server
* Performed data transformation using **Power Query**
* Built interactive dashboard with:

  * KPI cards
  * Trend charts
  * Category and size analysis
  * Navigation buttons for better UX

---

## 📊 Key Insights

* **Total Revenue:** $817,860
* **Total Orders:** 21,350
* **Total Pizzas Sold:** 49,574
* **Average Order Value:** $38.30
* **Average Pizzas per Order:** 2.32
* **Large Size Pizza Contribution:** 45.89%

---

## 💡 Business Insights

* Identified **peak sales periods** (daily & monthly trends)
* Highlighted **best and worst performing pizzas**
* Analyzed **customer ordering patterns**
* Provided actionable insights for:

  * Inventory planning
  * Menu optimization
  * Sales strategy
